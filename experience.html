Experience

To find out more about my individual experiences please click the relevant logo!

Towards the end of 2020 I was inspired to join a coding course for women put together by SheCodes, focusing on Front end development. As part of this course, I completed 4 projects (including this one!) each ending in their own website.

  • First, I created a travel page where I learned the basics of HTML, CSS and JS, creating a simple website with some custom content and css. To apply my first JavaScript skills, I implemented a button with a pop up asking a series of questions and then updated the HTML, depending on the answers. This was a brilliant project to get to grips with how the different headings, paragraphs, links and elements such as span, div, strong, buttons and images interact with each other and figuring out best practices of how to target classes and elements with their specific css.
  • Second, I created a basic weather app, where a user can input the name of the city or use their current location to get the current weather and the forecast for the next week. The user can change the forecast into °F and back to °C. I was able to put a lot of the things I learnt for my travel page into practise and learned a whole host of news skills! I learned how to use bootstrap for easy styling using grids and the power of bootstrap classes, which enable you to change the look of an element by just adding a single word to its html. I used an API (and found out what that is!) for the first time, learning how to link it to my code, how to execute an API call and how to then manipulate the result it gives me to show the desired information on the screen. Using more complicated JavaScript functions, I learned how to use parameters and how to update bigger pieces of my website in one go. Finally, I learned how to push my code to github and how to host my site on an actual website.
  • For my third project, I created a new and improved weather app using React. I started with the same basic layout of my first weather app and transferred it into React. I then learned how to create my HTML fully within JavaScript and create components so I could minimise duplication in the code. I learned about nested components and how to send information from one component to another using parameters. I managed to expand on my original forecast and add an hourly forecast for each day when that day is clicked
  • This website is my final project and was all about responsive coding. I created a personal portforlio page to showcase my experience and CV to potential future employers which is optimised for all devices. I learned how to host a multi-page website and how to navigate between the pages. Using advanced bootstrap skills such a flexbox and resposive grids, I could make sure all the elements reacted properly on smaller screen sizes and using media queries I was able to change or hide certain parts of the website based on the screen size. For example, if you are reading this on a laptop, feel free to decrease the window size and watch as parts of the headings disappear and the logos change at different breakpoints!

All my projects are opensourced and the code can be found on my Github profile. I'm still learning so if you have any tips or suggestions for improvements I'd love to hear them!
